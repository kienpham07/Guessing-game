## Description

A lightweight browser-based number guessing game built with vanilla HTML, CSS, and JavaScript. Players try to guess a randomly generated number between 1 and 20, while the game tracks their score and high score.

## Features

- ✅ Random number generation (1–20)
- ✅ Scoring system with high score tracking
- ✅ Instant feedback (“Too high”, “Too low”, “Correct!”)
- ✅ Reset button to play again
- ✅ Animated emoji effects on win / loss

## Tech Stack

- **Languages:** HTML, CSS, JavaScript
- **Frameworks / Libraries:** None (vanilla JS)
- **Tools:** Any modern browser (no build tools required)

## Usage

1. Open `index.html` in a browser.
2. Enter a number between 1 and 20 in the input field.
3. Click **Check!** to submit your guess.
4. Watch the score drop with each incorrect guess.
5. Click **Again!** to reset the game and start a new round.
